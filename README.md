# Expense Tracker

## Server Setup

### 1. Install Dependencies:

```bash
bun install
```

### 2. Run the Server:

```bash
bun run dev
```

## Frontend Setup

### 1. Navigate to the Frontend Directory:

```bash
cd ./frontend/
```

### 2. Install Dependencies:

```bash
bun install
```

### 2. Run the Frontend:

```bash
bun run dev
```
